## ‚ú® Key Features

### üèóÔ∏è **Architectural Studio**
- **Precision Floor Plan Editor**: Drag-and-drop walls, doors, and windows with auto-snapping grid systems (20px precision).
- **Undo/Redo Time Travel**: Powerful history stack allowing strictly linear state traversal for error-free editing.
- **Unified Studio Panel**: A consolidated "Cockpit" interface on the right, managing everything from tool selection to property customization.

### üé® **Premium Visuals**
- **Glassmorphism UI**: A deep, dark-themed interface with real-time backdrop blurs, gradients, and subtle noise textures.
- **Dynamic Lighting**: Ambient glow effects and micro-interactions that respond to user input.
- **Responsive Canvas**: Infinite panning and zooming powered by `react-konva`.

### üß† **Intelligent State**
- **Zustand Architecture**: Global state management handling complex relationships between floors, projects, and furniture.
- **Local Persistence**: Automatic saving to local storage ensures you never lose a pixel of progress.

## üöÄ Getting Started

Clone the repository and ignite the engine:

```bash
# 1. Clone the repository
git clone https://github.com/Mehulbirare/quantumplan.git

# 2. Enter the quantum realm
cd quantumplan

# 3. Install dependencies
npm install

# 4. Ignite the dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to witness the future.
